# Projeto de Extensão FAMETRO 2026.1

Landing page institucional estática do Projeto de Extensão FAMETRO 2026.1, do Prof. Abraão Henrique S. Rosal.

## Estrutura

- `index.html`: conteúdo e seções da página
- `style.css`: layout, responsividade e identidade visual
- `script.js`: menu mobile, FAQ expansível e destaque de navegação
- `ImagensNovas/`: imagens institucionais usadas na página
- `materiais/`: documentos oficiais e modelos para acesso dos alunos

## Edição rápida

- Imagens: substitua `ImagensNovas/Unifametro.png`, `ImagensNovas/1.jpeg`, `ImagensNovas/2.jpg` e `ImagensNovas/logoGemini.png`.
- Materiais: substitua os arquivos em `materiais/` mantendo os nomes normalizados para não quebrar os links.
- WhatsApp: altere o `href` do link com a classe `whatsapp-button` e do link `support-link` do card "Grupo geral de WhatsApp" no `index.html`.
- Gem de dúvidas: altere o `href` do link com a classe `floating-gem` no `index.html`.
- Cronograma: atualize os cards e a tabela da seção `Principais Datas 2026.1` e troque o PDF oficial em `materiais/cronograma-central-extensao-curricular-2026-1.pdf`.
- Regras: atualize a seção `Comprovação, Envio e Avaliação` e o arquivo `materiais/regras-cec-2026.png`.
- FAQ: edite diretamente as perguntas e respostas dentro da seção `#faq`.

## Publicação no GitHub Pages

1. Envie estes arquivos para um repositório GitHub.
2. No GitHub, acesse `Settings` > `Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Escolha a branch principal e a pasta `/root`.
5. Aguarde a URL pública ser gerada.

O projeto não usa backend nem etapa de build.
